# HRMS Mobile Application

Flutter mobile application for the Human Resources Management System with NFC attendance tracking.

## Status

**Source Code**: ✅ Complete  
**Build Scripts**: ✅ Provided  
**Platform Directories**: ⚠️ Require generation (see below)  
**APK Build**: ✅ Script available after platform generation

## Quick Start

### 1. Generate Platform Directories
```bash
cd mobile
chmod +x setup-mobile.sh
./setup-mobile.sh
```

### 2. Build APK
```bash
./build-apk.sh
```

## Prerequisites

- Flutter SDK 3.16.0+
- Android Studio (for Android builds)
- Xcode (for iOS builds, macOS only)

## Features

- **NFC Attendance**: Scan NFC cards for clock in/out
- **Authentication**: JWT-based login with role-based access
- **Employee Dashboard**: View attendance, leave requests, payroll
- **Offline Support**: Basic offline functionality
- **Push Notifications**: Leave approvals, attendance alerts

## Configuration

Set API base URL via environment:
```bash
# Build with custom API URL
flutter build apk --dart-define=API_BASE_URL=https://your-api.com/api
```

## Build Output

APK location: `build/app/outputs/flutter-apk/app-release.apk`

## Platform Directories Note

Platform directories (`android/`, `ios/`) are **not committed** to the repository as they:
1. Are large (100+ MB)
2. Contain environment-specific paths
3. Are generated by Flutter tooling
4. Frequently updated

See [PLATFORM_SETUP.md](PLATFORM_SETUP.md) for detailed setup instructions.
