# 🏢 RABIE&RADOO معالج كشوف الرواتب

## 🚀 التشغيل السريع
```
شغل: RABIE_RADOO.lnk (مع أيقونة جميلة) 
أو: RABIE_RADOO.bat (عادي)
```

## 📁 محتويات المجلد
- **RABIE_RADOO.lnk** - المشغل المفضل (مع أيقونة الشركة)
- **RABIE_RADOO.bat** - المشغل الأصلي
- **app/** - ملفات البرنامج
- **tools/** - أدوات المساعدة والإصلاح

## 🎨 الأيقونة المخصصة
```
✅ متوفرة في RABIE_RADOO.lnk
🎯 حرف R بألوان الشركة
✨ مظهر احترافي
```

## 🛠️ في حالة المشاكل
```
شغل: tools\START_HERE.bat
أو راجع: tools\ للأدوات المساعدة
```
