## 🚀 طريقة الاستخدام

**انقر نقراً مزدوجاً على** `run_gui.bat`

## 📁 الملفات الأساسية:
- `input-master.xlsx` - ملف البيانات (243 موظف)
- `multi_sheet_payroll_processor.py` - محرك المعالجة  
- `payroll_gui.py` - الواجهة الرسومية المحدثة
- `README.md` - دليل الاستخدام
- `requirements.txt` - المتطلبات
- `run_gui.bat` - ملف التشغيل

## 📋 خطوات العمل:
1. **افتح البرنامج** - انقر على `run_gui.bat`
2. **اختر ملف الإدخال** - تصفح لـ `input-master.xlsx`
3. **اختر مجلد الحفظ** - حدد أين تريد حفظ النتائج 📁
4. **اختر الشهر والعام** - من القوائم المنسدلة
5. **اكتب ملاحظات** (اختياري) - ستظهر في جميع الكشوف  
6. **اضغط START** 🚀 - وانتظر النتيجة!

## ⚠️ متطلبات التشغيل:
- **Python 3.7+** (مع tkinter مدمج)
- **Microsoft Excel** (لتحويل PDF)
- **المكتبات:** `pip install -r requirements.txt`

## 📦 نقل البرنامج لكمبيوتر آخر:
1. **انسخ مجلد البرنامج كامل**
2. **فحص تلقائي:** انقر على `check_requirements.bat` 
3. **شغل البرنامج:** انقر على أحد هذه الملفات:
   - `run_gui_silent.vbs` ← **مُوصى به**: يشغل البرنامج بدون إظهار موجه الأوامر
   - `run_gui.bat` ← يشغل البرنامج ويقفل موجه الأوامر تلقائياً

### 🛠️ الطريقة اليدوية:
1. **تأكد من Python:** `python --version`  
2. **ثبت المتطلبات:** `pip install -r requirements.txt`  
3. **شغل البرنامج:** انقر على `run_gui_silent.vbs` للتشغيل الصامت
