# 🎵 مجلد الأصوات - RABIE&RADOO

## 📁 الملفات المطلوبة:

ضع الملفات الموسيقية التالية في هذا المجلد:

### **1. صوت زر التحقق:**
- **الاسم المطلوب**: `check_sound.mp3` أو `check_sound.wav`
- **الوصف**: يُشغل عند الضغط على زر "✅ تحقق"
- **المدة المقترحة**: 10-30 ثانية (يتكرر أثناء الفحص)

### **2. صوت زر البدء:**
- **الاسم المطلوب**: `start_sound.mp3` أو `start_sound.wav`
- **الوصف**: يُشغل عند الضغط على زر "🚀 START"
- **المدة المقترحة**: 10-30 ثانية (يتكرر أثناء المعالجة)

## 🎼 **الصيغ المدعومة:**
- `.mp3` - مُفضل
- `.wav` - جودة عالية
- `.ogg` - بديل جيد

## 📏 **المواصفات المقترحة:**
- **الحجم**: أقل من 2 MB لكل ملف
- **الجودة**: 44.1 kHz كافية
- **الطول**: 10-30 ثانية (للتكرار المريح)

## 🔄 **ميزة التكرار الجديدة:**
- 🎵 **الأصوات تتكرر تلقائياً** أثناء المعالجة
- 🔇 **تتوقف تلقائياً** عند انتهاء العملية
- ⏹️ **يمكن إيقافها** عند إغلاق البرنامج أو النقر على زر آخر

## 📂 **البنية النهائية:**
```
sounds/
├── check_sound.mp3    ← ضع ملف صوت التحقق هنا
├── start_sound.mp3    ← ضع ملف صوت البدء هنا
└── README.md         ← هذا الملف
```

## 🚀 **كيفية العمل:**
1. **عند الضغط على "✅ تحقق"** → يبدأ الصوت ويتكرر أثناء الفحص
2. **عند الضغط على "🚀 START"** → يبدأ الصوت ويتكرر أثناء المعالجة  
3. **عند الانتهاء** → يتوقف الصوت تلقائياً
4. **عند إغلاق البرنامج** → تتوقف جميع الأصوات
